Direction<String> class

Possible return values for {@link HasDirection#getDirection()} and parameter values for {@link HasDirection#setDirection(Direction)}.Widgets that implement this interface can either have a direction that is right-to-left (RTL), left-to-right (LTR), or default (which means that their directionality is inherited from their parent widget).
